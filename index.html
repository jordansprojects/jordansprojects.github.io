ChAIckers 2 (In Progress)

Checker playing AI inspired by Alpha Zero

ChAIckers was a project for a Deep Learning course at my University. We successfully created a Deep Learning model that can play checkers, but the final result did not utilize the Monte Carlo Tree Search algorithm like we had originally intended. The move generation was too slow, so it was impractical trying to include it. Unsatisfied with the original result, I wanted to refactor it by improving upon the move generation code, writing my own monte carlo tree search library, and redoing the deep learning model so that it more closely resembles the Alpha Zero agent.
