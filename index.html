About Me

I was working in auto repair and intended for that to be my career. I took a C++ programming course and came to the realization that programming is extremely fun. I decided to pursue a degree in Computer Science. Along the way, I realized how fascinating Mathematics is- so I decided to minor in it. I enjoy problem solving and puzzles, and I think it is amazing that we can automate machines to solve them for us.

On my YouTube Channel, I like to post clips of my projects. Some of them are simply little programs I was making to learn a skill, and others are larger projects.

I created a playlist that demonstrates more completed works that I have created for school assignments. I always want to improve upon my projects, so I have a difficult time ever considering them fully complete.

Here are some of my projects

On The Farm : Virtual Reality Tower Defense Game

Implemented in Unity Game Engine using Steam VR

This game was created as a final project for a Virtual Reality class at my university. We implemented this game using C# scripts, Emerald AI and VRIF

8-Puzzle Solver & Game

An Interactive 8-Puzzle Game and Solver with an Animated Goblin

For a data structures class in community college, we had to create a project using the graph data structure. Luckily, graph theory is incredibly interesting and we were barely scratching the surface in this course. This program utilizes a graph search to solve 8-Puzzle.

ChAIckers 2 (In Progress)

Checker playing AI inspired by Alpha Zero

ChAIckers was a project for a Deep Learning course at my University. We successfully created a Deep Learning model that can play checkers, but the final result did not utilize the Monte Carlo Tree Search algorithm like we had originally intended. The move generation was too slow, so it was impractical trying to include it. Unsatisfied with the original result, I wanted to refactor it by improving upon the move generation code, writing my own monte carlo tree search library, and redoing the deep learning model so that it more closely resembles the Alpha Zero agent.
